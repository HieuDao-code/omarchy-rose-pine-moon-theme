# Rose pine moon theme for Omarchy

A rose pine moon theme for [Omarchy](https://omarchy.org/).

## Preview

![omarchy-rose-pine-moon-theme](preview.png)

## Installation

Use the builtin installation script (available after version 1.4.0):

```bash
omarchy-theme-install https://github.com/HieuDao-Code/omarchy-rose-pine-moon-theme
```
